REQUISITOS DE USUARIO:

Desejável
{
    Lista com livros já lidos
    Lista com futuras leituras
    Um fórum pra cada livro
    O usuário pode requerer livros novos

    Importante
    {
        Dar nota aos livros
        Ter pelo menos um fórum geral para comentários sobre os livros
        Ter um usuário bibliotecário
        Ter um usuário moderador, para controlar o fórum
        Ter um usuário administrador
        Cada usuário ter seu perfil

        Essencial
        {
            O usuario podera ler livros
            O usuário devera ser registrado
        }
    }
}
